# Sootudy

15기 서울 1반 스터디 저장소입니다.

## 📋 프로젝트 소개

15기 서울 1반 팀원들이 스터디 내용을 공유하고 기록하는 협업 저장소입니다.

## 🎯 프로젝트 목표

- 팀원 간 학습 내용 공유
- 스터디 기록 관리
- 상호 피드백을 통한 학습 품질 향상

## 📁 디렉토리 구조

```
Sootudy/
├── README.md
├── 01_howTo/
├── templates/
├── [팀원이름]/
│   └── 주제/
│       └── 내용.md
└── ...
```
본인 이름(깃허브 아이디)으로 폴더를 만들고 그 안에 스터디 내용을 작성해주세요!
